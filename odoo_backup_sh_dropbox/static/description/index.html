Backups into Dropbox

Manage database backups via Dropbox

Main feature: no need to download and upload a database archive via a browser.

Configuration


Before starting backup jobs you need to make some configuration.
Note: You need to have a Dropbox account
  • Creating Access Token
    • - Open the App Console https://www.dropbox.com/developers/apps
    - Login/Register into Dropbox account and click Create app
    - Choose an API (Dropbox API) and type of access (App folder)
    - Specify the name of your App (Odoo Backups)
    - Read and accept the agreement, then confirm by clicking Create app
    - Click to Generate button in order to Generated access token and Save it
  • Folder in Dropbox
    • - Open your Dropbox homepage https://www.dropbox.com/home
    - Open Applications → Odoo Backups
    - Create new folder Backups
  • Config Settings
    • - Activate Developer Mode https://odoo-development.readthedocs.io/en/latest/odoo/usage/debug-mode.html
    - Open menu Backups → Configuration → Settings
    - Choose Dropbox option and press Save
    - Specify Dropbox Access Token
    - Specify your folder path /Backups and press Save

How it works

Go to Backup Dashboard and press Add button

Make configuration depending on your needs

  • Define the Name
  • Create the Password if you want to encrypt the Backup
  • Schedule Auto Backups
  • Set up how many backups to preserve for certain time frames

You can see details of your backups in the Table format

You can see a Visual representation of your backups
